# Invernadero_arduino

En este repositorio, el código principal del ESP-32 se encuentra en la subcarpeta:

```
Proyecto_Invernadero_ESP32_Adafruit_IO/
  └── Proyecto_Invernadero_ESP32_Adafruit_IO/
        └── src/
              └── main.cpp
```

- **Proyecto_Invernadero_ESP32_Adafruit_IO/src/main.cpp:**  
  Aquí está el código fuente que debes subir/flashear al ESP-32.
- El resto de archivos incluye documentación, materiales complementarios y otros recursos del proyecto.

## Estructura del proyecto

```
Proyecto_Invernadero_ESP32_Adafruit_IO/
│
├── Proyecto_Invernadero_ESP32_Adafruit_IO/
│   └── src/
│       └── main.cpp           # Código principal del ESP-32
│
├── README.md                  # Este archivo
├── (otros archivos y carpetas)
```

## Notas

- Para modificar el código del ESP-32, edita el archivo:  
  `Proyecto_Invernadero_ESP32_Adafruit_IO/Proyecto_Invernadero_ESP32_Adafruit_IO/src/main.cpp`
- Si usas Arduino IDE o PlatformIO, selecciona el archivo `main.cpp` mencionado para flashear tu dispositivo.
